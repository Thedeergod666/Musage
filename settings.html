🟧 Xiaomi MiMo Token Plan Key

—
🔒 Token Plan 专用 key（tp- 开头，区别于 pay-as-you-go 的 sk-）。
从 platform.xiaomimimo.com 订阅后获取。

🍪 Dashboard Cookie（用量查询用）

—
⚠️ Xiaomi 用量走 dashboard admin API，需要浏览器登录态。
获取方法：Chrome 登录 platform.xiaomimimo.com → F12 → Network → 任意 /api/v1/tokenPlan/* 请求 → 右键 → Copy → Copy request headers → 找 cookie: 这一行整段粘贴到上面。
Cookie 登出后失效，过期时 (HTTP 401) 错误信息会引导重粘。
集群字段当前用于未来扩展（多账号 / region-specific URL）。当前 cookie auth 已绑定 user，无视。
🔧 Schema overrides（高级 · MiniMax 改字段名后填这里）
只在 解析失败 / 显示 "Schema 未知" 时用。每行一个 JSON 对象，total + remaining 同时存在才算命中，end 可选。 改完点"保存配置"。参考 ccswitch 的 schema 字段名逆向。
最少 10 秒，默认 60。对所有 provider 同时生效。

🪟 浮窗

改动会立即生效（不需要点"保存配置"），并写入 config.json 下次启动恢复。
浮窗位置 + 大小已经自动记忆，下次启动恢复。点"归位"可把窗口挪到当前主屏幕正中央。
老 Intel Mac / Linux WebKitGTK 等 GPU 弱的平台开启可显著减少 backdrop-filter 开销。 视觉上 idle 状态会丢失玻璃质感，hover 仍保留色彩切换。改动会立即生效。
检测到任何 app 进入全屏（菜单栏自动隐藏）→ 浮窗暂时隐藏；退出全屏 → 自动恢复。 目前仅 macOS 生效（通过 NSMenu.menuBarVisible 探测）。
⚠ 若你在「系统设置 → 桌面与程序坞 → 在桌面上自动隐藏并显示菜单栏」开了，可能会被误触发。
